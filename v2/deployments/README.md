# Deployments

Artefactos de despliegue y runtime del proyecto.

## Contenido

- `blockchain_contract.json`: bridge activo del contrato.
- `blockchain_contract.localhost.json`: artefacto de local.
- `blockchain_contract.sepolia.json`: artefacto de Sepolia.
- `runtime/issuer_wallet.json`: wallet issuer local.
- `runtime/wallet.json`: wallet holder local.
- `runtime/sepolia_deployer_wallet.json`: wallet deployer testnet.
- `runtime/ssi_sessions.db`: base de datos sqlite local.

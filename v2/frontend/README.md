# Frontend

Interfaces web y servidor estatico seguro.

## Archivos

- `frontend_portal.html`: portal de entrada.
- `issuer_dashboard.html`: dashboard de emisión y revocación.
- `verifier_dashboard.html`: dashboard de verificación.
- `frontend.variables.js`: variables de UI generadas por `scripts/start_all.py`.
- `frontend_server.py`: servidor estatico con whitelist.

# Documentacion v2

Este directorio organiza la documentacion del proyecto por finalidad.

## Estructura

- `indices/`: mapas del repositorio y rutas de lectura recomendadas.
- `tutoriales/`: guias paso a paso para operar y usar el sistema.
- `referencia/`: decisiones tecnicas, perfiles y notas de arquitectura.
- `operacion/`: resumen consolidado del estado operativo.

## Punto de entrada recomendado

1. `indices/README.md`
2. `indices/REPO_INDEX.md`
3. `tutoriales/README.md`
4. `referencia/README.md`

## Lectura rapida por objetivo

- Quiero desplegar en testnet: `tutoriales/TESTNET_QUICKSTART.md`
- Quiero entender frontends separados: `tutoriales/FRONTENDS_SPLIT_GUIDE.md`
- Quiero entender decisiones del codigo: `referencia/GUIA_CODIGO_Y_CAMBIOS.md`

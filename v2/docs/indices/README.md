# Indices

Carpeta para navegar rapido el repositorio.

## Contenido

- `REPO_INDEX.md`: inventario de archivos por area (arranque, backend, frontend, blockchain, tests, docs).

## Uso recomendado

1. Abrir `REPO_INDEX.md`.
2. Ir a la seccion del componente que necesitas modificar.
3. Saltar a `docs/tutoriales` o `docs/referencia` para detalle funcional/arquitectonico.
